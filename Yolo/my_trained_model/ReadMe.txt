Можно запустить в гугл коллабе. Данная модель с названием best.pt тренированна на основе датасета созданного мной. Запускать каждый блок по очереди. 

На вывод вы получите видео на которым модель будет определять класс Car_plate(номер машины). (Если запускать в коллабе с использованием видеокарты, то процесс будет в 100 раз быстрее)
